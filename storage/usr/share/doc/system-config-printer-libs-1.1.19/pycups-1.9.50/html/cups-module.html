setPasswordCB(fn)

 

Set password callback function. This Python function will be called when a password is required. It must take one string parameter (the password prompt) and it must return a string (the password). To abort the operation it may return the empty string ('').

Parameters:
  • fn (callable object) - callback function
Returns: None

setPasswordCB2(fn, context=None)

 

Set password callback function. This Python function will be called when a password is required. It must take parameters of type string (the password prompt), instance (the cups.Connection), string (the HTTP method), string (the HTTP resource) and, optionally, the user-supplied context. It must return a string (the password). To abort the operation it may return the empty string ('').

Parameters:
  • fn (callable object, or None for default handler) - callback function
Returns: None
