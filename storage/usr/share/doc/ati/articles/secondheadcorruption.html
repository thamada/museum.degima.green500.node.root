Second head in Dual Head system corrupt during login screen

This information applies to the following system configurations:

  • ATI Proprietary Linux Driver version 3.14.6 and earlier
  • ATI graphics adapters with Dual Head capability
  • Dual Head X Server configuration
  • system configured for Graphical Login Prompt

Screen with Graphical Login Prompt appears normally on primary display.  The secondary display appears corrupt.

This issue has been resolved in the 8.8.25 driver.
