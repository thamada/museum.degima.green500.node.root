ATI Linux Driver 3.7.1 - X may hang or fail to start on any system with an Athlon CPU

This issue affects the following system configurations:

  • ATI Proprietary Linux driver version and 3.7.1
  • XFree86 4.1.0, 4.2.0, and 4.3.0
  • AMD Athlon based CPU

The system may hang while launching starting X.

To resolve this issue, download and install Display Driver version 3.7.6 or higher.
