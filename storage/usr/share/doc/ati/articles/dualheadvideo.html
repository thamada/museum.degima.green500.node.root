Video playback applications in Dual Head mode produce an error message or fail to start

This information applies to the following system configurations:

  • ATI Proprietary Linux Driver version 3.14.6 and earlier
  • ATI display adapter with Dual Head capability
  • X Server Dual Head configuration

The following may occur during the execution of a video playback application:

  • fails to start
  • video playback error
This does not apply to Single Head X Server configurations.

This issue has been resolved in the 8.8.25 driver.
