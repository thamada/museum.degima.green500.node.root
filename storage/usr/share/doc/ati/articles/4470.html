Linux - Hardware acceleration is not available using applications in WineX

This issue affects the following configurations:

  • ATI proprietary Linux driver version 3.7.0 and 3.7.1
  • Xfree86 4.1.0, 4.2.0, and 4.3.0

When applications are run under WineX, hardware acceleration will not be available.

To resolve this issue, download and install Display Driver version 3.7.6 or higher.
