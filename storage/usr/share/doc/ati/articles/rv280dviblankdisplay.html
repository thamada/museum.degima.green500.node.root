Blank display using DVI connector on some RADEON 9200 boards

This information applies to the following:

  • ATI Proprietary Linux Driver
  • RADEON 9200 family graphics cards with DVI connector

When launching X-Windows, any monitor connected to the DVI port will appear blank. The display appears fine for the text console, however. Monitors connected to the VGA connector function normally.

Not all RADEON 9200 cards are affected.

This issue has been resolved in ATI Proprietary Linux Driver version 3.14 and later.
