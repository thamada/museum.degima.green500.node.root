Random polygons appear throughout display while playing Doom 3

This information applies to the following system configurations:

  • ATI Proprietary Linux Driver 3.14.1 and earlier
  • Doom 3 Linux Client by ID Software

Doom 3 will launch properly, but the game menu will appear corrupt, as will gameplay. Alternately, the menu may appear normally, but the display will become completely corrupt shortly into normal gameplay.

Random polygons will appear all over the display. Keyboard and mouse control should not be affected.

This issue has been resolved in the 3.14.6 hotfix driver.
