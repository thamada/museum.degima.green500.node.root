Stopping and restarting the X Server no longer results in memory leaks or hangs on PCI-Express systems

This information applies to the following system configurations:

  • ATI Proprietary Linux Driver version 3.14.6 and earlier
  • PCI-Express Systems

Initially, the X Server launches properly.  Closing the X Server may result in a memory leak.  Subsequent attempts to launch the X Server may
result in a kernel panic or system hang.

This issue has been resolved in the 8.8.25 driver.
