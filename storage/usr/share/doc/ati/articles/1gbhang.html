3D Applications hang on RHEL 3.0 with 1GB or more system memory installed

This information applies to the following system configurations

  • ATI Proprietary Linux Driver 3.14.1 or earlier
  • Red Hat Enterprise Linux 3.0
  • 1GB or more system memory installed

After running a 3D application for some time, the system will lock up. SPEC viewperf is known to exhibit this behaviour. Other applications may also be affected.

This issue has been resolved in the 3.14.6 hotfix driver.
