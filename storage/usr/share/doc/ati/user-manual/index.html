ATI's Linux Driver

Features at a Glance

The following features are supported by ATI's High Performance Proprietary Driver. Certian features are not supported on some cards.

2D Features

  • Accellerated 3D
  • Multiple Monitors
  • Direct Graphics Access
  • Mixed 8/24 bpp Visuals

3D Features

  • Hardware Accellerated OpenGL
  • OpenGL 1.5
  • 3D Stereo
  • OpenGL Mixed Visuals
  • Full Scene Anti-Aliasing
  • Application Profiles

Multimedia Features

  • TV Out
  • XVideo Video Acceleration
