Detailed Description

Mixer Interface.

Author:
Jaroslav Kysela <perex@perex.cz>
Abramo Bagnara <abramo@alsa-project.org>
Date:
2001

Mixer interface is designed to access mixer elements. Callbacks may be used for event handling.
